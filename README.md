Data Analyst Assignment

Overview  
This project is a small collection of tasks that helped me practice core data analysis skills using SQL, Excel, and Python. It includes real-world scenarios like managing hotel bookings, analyzing clinic data, working with spreadsheets, and solving basic programming problems.

Project Structure  
The project is organized into three main parts.  
The SQL folder contains all database-related work for both hotel and clinic systems.  
The Spreadsheets folder includes the Excel file where ticket data analysis is performed.  
The Python folder contains scripts for solving simple logic-based problems.  

SQL Work  
In this section, I created tables and wrote queries to analyze data.  
For the hotel system, I worked on tasks like finding the last booked room, calculating billing amounts, and identifying popular items.  
For the clinic system, I analyzed revenue, customer data, and profit across different locations.  
I used concepts like joins, group by, aggregation, and ranking.

Spreadsheet Work  
In the spreadsheet section, I worked with ticket and feedback data.  
I used VLOOKUP to connect data between sheets and applied date and time functions to find patterns like tickets closed on the same day or within the same hour.  
I also used COUNTIF and COUNTIFS to summarize the data.

Python Work  
In Python, I solved two simple problems.  
One was converting minutes into a readable format of hours and minutes.  
The other was removing duplicate characters from a string using loops.

Tools Used  
I used MySQL or DB Fiddle for SQL queries.  
Excel or Google Sheets for spreadsheet tasks.  
Python 3 for programming.

Conclusion  
This assignment helped me understand how different tools are used together in data analysis. It also improved my problem-solving and logical thinking skills.
